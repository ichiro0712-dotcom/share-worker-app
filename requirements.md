# S WORKS - 要件定義書

## 1. サービス概要

### 1.1 サービス名
**S WORKS（エスワークス）**

### 1.2 サービス説明
看護師・介護士のための求人マッチングWebサービス。
単発・短期の仕事を中心に、働きたい日時で柔軟に仕事を探せるプラットフォーム。

### 1.3 ターゲットユーザー
- 看護師（正看護師、准看護師）
- 介護士（介護福祉士、ヘルパー等）
- ブランクがある方
- 副業・Wワークを希望する方
- 単発・短期で働きたい方

## 2. 主要機能

### 2.1 ユーザー機能

#### 2.1.1 会員登録・ログイン
- **会員登録**
  - メールアドレス + パスワード
  - SNS連携（Google、LINE）
  - 基本情報登録（氏名、生年月日、電話番号）
  - 資格情報登録（看護師、介護福祉士等）
  - プロフィール写真（任意）

- **ログイン**
  - メールアドレス + パスワード
  - SNS連携ログイン
  - ログイン状態の保持

- **プロフィール管理**
  - 基本情報編集
  - 資格・経験年数の編集
  - 希望勤務条件の設定
  - 通知設定

#### 2.1.2 求人検索・閲覧

- **トップページ（求人一覧）**
  - 全体/限定/指名のタブ切り替え
    - 全体：すべての求人
    - 限定：ユーザー限定の特別求人
    - 指名：施設から指名された求人

- **検索フィルター機能**
  - 働ける日（カレンダー選択）
  - 勤務時間帯（朝、昼、夜、夜勤）
  - 施設種別（デイサービス、特養、有料老人ホーム等）
  - 職種（看護、介護）
  - エリア（都道府県、市区町村、駅）
  - 時給・日給範囲
  - 資格要件
  - その他条件（駅チカ、未経験OK、高時給等）

- **並べ替え機能**
  - 近い順（位置情報ベース）
  - 時給順（高い順）
  - 締切が近い順
  - 新着順

- **日付スライダー**
  - 今日から3ヶ月分の日付を横スクロール表示
  - 日付選択で該当日の求人を表示

- **求人カード表示**
  - 施設画像
  - 施設名
  - 施設評価（星5段階＋レビュー数）
  - 求人タイトル
  - 勤務時間
  - 報酬（日給・時給）
  - 締切時間（カウントダウン）
  - 住所・アクセス
  - タグ（施設種別、特徴）
  - ブックマーク機能

#### 2.1.3 求人詳細

- **基本情報**
  - 勤務日時
  - 施設名・住所
  - 地図表示（Google Maps連携）
  - アクセス方法（交通手段、駐車場の有無）
  - 施設画像（複数枚、カルーセル表示）

- **募集情報**
  - 募集人数／応募状況
  - 職種・業務内容
  - 必要資格
  - 経験・スキル要件
  - 報酬（時給、日給、交通費）
  - 勤務時間・休憩時間
  - 締切日時

- **仕事内容詳細**
  - 業務内容の詳細説明
  - 仕事内容タグ（移乗介助、バイタル測定等）
  - 施設の特徴・雰囲気

- **事前情報**
  - 服装・持ち物
  - その他条件
  - 法人情報（法人名、電話番号）

- **レビュー**
  - 総合評価（星5段階）
  - 評価分布（5〜1の件数）
  - 個別レビュー
    - レビュアー情報（年代、性別、職種、勤務期間）
    - 良かった点
    - 改善点
  - レビュー件数表示
  - もっと見る機能

- **責任者情報**
  - 担当者名
  - 担当者メッセージ
  - 担当者アバター

- **その他の応募日時**
  - 同じ施設の別日程の求人表示
  - 横スクロールで複数表示

- **インタラクション**
  - お気に入り登録
  - あとで見る保存
  - ミュート機能（この施設の求人を表示しない）
  - タブナビゲーション（仕事概要、申込条件、事前情報、レビュー）

#### 2.1.4 応募機能

- **応募フロー**
  1. 求人詳細ページで「申し込む」ボタンをクリック
  2. 応募確認画面（プロフィール、資格確認）
  3. 応募完了
  4. 完了画面表示
  5. 確認メール送信

- **応募管理**
  - 応募中の求人一覧
  - 応募履歴
  - ステータス管理
    - 応募中
    - 承認待ち
    - 採用決定
    - 不採用
    - キャンセル

#### 2.1.5 下部ナビゲーション

- **探す**（トップページ）
  - 求人検索・一覧表示

- **お気に入り**
  - Phase 2以降で実装（現在は工事中ページ表示）

- **メッセージ**
  - Phase 2以降で実装（現在は工事中ページ表示）
  - 未読バッジ表示機能あり

- **仕事管理**
  - Phase 2以降で実装（現在は工事中ページ表示）

- **マイページ**
  - Phase 2以降で実装（現在は工事中ページ表示）

### 2.2 施設（求人掲載側）機能

#### 2.2.1 施設アカウント登録
- 法人情報登録
- 施設情報登録
- 担当者情報登録
- 本人確認（書類審査）

#### 2.2.2 求人掲載
- 求人情報入力
  - 勤務日時
  - 募集人数
  - 職種・業務内容
  - 必要資格・経験
  - 報酬
  - 交通費
- 施設情報・写真登録
- 掲載期間設定
- プレビュー・公開

#### 2.2.3 応募者管理
- 応募者一覧
- 応募者プロフィール確認
- 採用/不採用の連絡
- メッセージ送信

#### 2.2.4 スタッフ管理
- 勤務実績記録
- 評価・レビュー
- 指名機能（リピート依頼）

#### 2.2.5 限定求人
- 特定ユーザーへの限定公開
- 条件を満たすユーザーへの自動公開
  - 過去の勤務経験あり
  - 高評価ユーザー
  - 特定資格保持者

### 2.3 管理者機能

#### 2.3.1 ユーザー管理
- ユーザー一覧
- ユーザー詳細確認
- アカウント停止・削除
- 資格情報確認

#### 2.3.2 施設管理
- 施設一覧
- 施設審査（新規登録時）
- 施設情報確認・編集
- アカウント停止・削除

#### 2.3.3 求人管理
- 求人一覧
- 求人内容確認
- 不適切な求人の削除・非公開
- 求人統計

#### 2.3.4 レビュー管理
- レビュー一覧
- 不適切なレビューの削除
- レビュー統計

#### 2.3.5 売上・手数料管理
- 取引履歴
- 手数料計算・請求
- 支払い管理

## 3. 非機能要件

### 3.1 パフォーマンス
- ページ読み込み時間：3秒以内
- 同時アクセス数：10,000ユーザー
- データベースレスポンス：100ms以内

### 3.2 セキュリティ
- SSL/TLS通信の暗号化
- パスワードのハッシュ化
- 個人情報の適切な管理
- XSS、CSRF対策
- SQLインジェクション対策

### 3.3 可用性
- 稼働率：99.9%以上
- バックアップ：1日1回
- 障害時の復旧時間：4時間以内

### 3.4 保守性
- ログ記録・監視
- エラー通知
- 定期的なセキュリティアップデート

### 3.5 対応環境
- **ブラウザ**
  - Chrome（最新版）
  - Safari（最新版）
  - Firefox（最新版）
  - Edge（最新版）
- **デバイス**
  - PC（1280px以上）
  - タブレット（768px〜1279px）
  - スマートフォン（767px以下）
- **レスポンシブ対応必須**

## 7. ビジネスモデル

### 7.1 収益源
- 施設からの掲載手数料
  - 求人掲載料：無料
  - 成約手数料：採用1件あたり◯◯円
  - または時給の◯%

### 7.2 ユーザーへの提供価値
- 無料で求人検索・応募可能
- 柔軟な働き方の実現
- 簡単な応募フロー
- 施設の評価・レビューで安心

### 7.3 施設への提供価値
- 短期・単発の人材確保
- 優良なスタッフの確保
- 簡単な求人掲載
- 応募者管理の効率化
