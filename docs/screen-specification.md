## 2. 各画面の詳細仕様

### 2.1 トップページ（求人一覧）

#### 画面構成

```
┌─────────────────────────────┐
│ ヘッダー                     │
│ ├ タブ（全体/限定/指名）      │
│ ├ フィルターエリア            │
│ │  ├ 働ける日ボタン          │
│ │  ├ 絞り込みボタン          │
│ │  └ 並び順ボタン            │
│ └ 日付スライダー              │
├─────────────────────────────┤
│ 求人カード1                  │
├─────────────────────────────┤
│ 求人カード2                  │
├─────────────────────────────┤
│ 求人カード3                  │
├─────────────────────────────┤
│ ...                         │
├─────────────────────────────┤
│ 下部ナビゲーション            │
└─────────────────────────────┘
```

#### 2.1.1 ヘッダーエリア

**タブ**
- 3つのタブ：「全体」「限定」「指名」
- 選択中のタブは緑色（#66cc99）でアンダーラインあり
- 未選択のタブはグレー（#6b7280）
- 初期状態：「全体」が選択状態

**フィルターエリア**
- 左側：「働ける日」ボタン
  - アイコン：📅
  - テキスト：「働ける日」
  - クリックでカレンダー表示（Phase 2）

- 右側
  - 「絞り込み」ボタン
    - アイコン：🔍
    - テキスト：「絞り込み」
    - クリックでフィルター画面へ（Phase 2）

  - 「並び順」ボタン
    - アイコン：🔍
    - テキスト：「並び順」
    - 下向き矢印：▼
    - クリックでドロップダウンメニュー表示
    - メニュー項目：
      - 近い順（デフォルト選択）
      - 時給順

**日付スライダー**
- 横スクロール可能
- 今日から90日分（3ヶ月）の日付を表示
- 各日付アイテムの表示：
  - 今日：緑背景（#66cc99）、白文字、「今日」と表示
  - 今日以外：グレー背景（#f3f4f6）
  - 選択中：薄緑背景（#e6f7f0）、緑枠線（#66cc99）
- 日付フォーマット：
  - 今日：「今日」
  - それ以外：「MM/DD」と曜日（例：「11/16」「土」）
- 初期状態：「今日」の次の日付（11/16）が選択状態

#### 2.1.2 求人カード

**カード構成**
```
┌─────────────────────────────┐
│ 施設画像                     │
│ ├ タグ（左上）               │
│ └ ブックマークアイコン（右上）│
├─────────────────────────────┤
│ 施設名 ★評価点 (レビュー数)  │
│ 求人タイトル（2行まで）       │
│ 勤務時間        日給         │
│                (時給)        │
│           締切まで XX:XX:XX  │
│ 住所                         │
│ アクセス                     │
└─────────────────────────────┘
```

**画像エリア**
- サイズ：横幅100%、高さ128px
- 画像なしの場合：プレースホルダー画像表示
- 左上：タグ（複数可、横並び）
  - 背景色：緑（#66cc99）
  - 文字色：白
  - フォントサイズ：12px
  - 例：「デイサービス」「介護」
- 右上：ブックマークアイコン
  - 未保存：白枠のみ
  - 保存済み：青塗りつぶし
  - クリックでブックマーク保存/解除（Phase 2）

**施設情報行**
- 施設名：太字、14px
- 評価：黄色い星（★）+ 評価点（例：4.9点）
- レビュー数：グレー文字（例：(330件)）
- フォントサイズ：12px

**求人タイトル**
- フォントサイズ：14px
- 最大2行表示、それ以上は「...」で省略

**勤務時間・報酬行**
- 左側：勤務時間（グレー、14px）
  - 例：「09:00-16:30」
- 右側：報酬
  - 日給：赤文字（#ef4444）、太字、18px
    - 例：「9,250円」
  - 時給：グレー文字、12px、括弧付き
    - 例：「(時給1,300円)」

**締切表示**
- 右寄せ
- 赤背景（#ef4444）、白文字
- フォントサイズ：12px
- 角丸
- 例：「締切まで22:42:22」

**住所・アクセス**
- グレー文字（#6b7280）、12px
- 2行表示
  - 1行目：住所（例：「東京都杉並区井草3-23-1F」）
  - 2行目：アクセス（例：「西武新宿線「下井草駅」より徒歩9分」）

**カード全体**
- 背景：白
- 枠線：薄グレー（#e5e7eb）
- 角丸：8px
- 影：軽いドロップシャドウ
- ホバー時：影を強調
- クリックで求人詳細ページへ遷移

#### 2.1.3 下部ナビゲーション

5つのメニューを等間隔で配置

```
┌────┬────┬────┬────┬────┐
│探す│お気│メッ│仕事│マイ│
│    │に入│セー│管理│ペー│
│    │り  │ジ  │    │ジ  │
└────┴────┴────┴────┴────┘
```

**各メニュー項目**
- アイコン（24x24px）
- ラベル（12px）
- 選択中：緑色（#66cc99）
- 未選択：グレー（#9ca3af）

**メニュー詳細**
1. **探す**
   - アイコン：検索（虫眼鏡）
   - ラベル：「探す」
   - 初期状態：選択中（緑色）
   - タップ時：トップページへ遷移

2. **お気に入り**
   - アイコン：ハート
   - ラベル：「お気に入り」
   - タップ時：工事中ページへ遷移

3. **メッセージ**
   - アイコン：吹き出し
   - ラベル：「メッセージ」
   - 未読バッジ：右上に赤丸（数字表示）
     - 背景：赤（#ef4444）
     - 文字色：白
     - サイズ：16x16px
     - 例：「3」
   - タップ時：工事中ページへ遷移

4. **仕事管理**
   - アイコン：クリップボード
   - ラベル：「仕事管理」
   - タップ時：工事中ページへ遷移

5. **マイページ**
   - アイコン：ユーザー
   - ラベル：「マイページ」
   - タップ時：工事中ページへ遷移

**ナビゲーション全体**
- 位置：画面下部固定
- 背景：白
- 上部に境界線（グレー、1px）
- 高さ：約60px
