## 各タスク対応手順

```
1. ブランチ作成: git checkout -b fix/{task-name}-{id}
2. 調査・実装
3. ビルド確認: npm run build
4. コミット・プッシュ
5. PR作成: gh pr create
6. マージ後: main に戻り、ブランチ削除
7. REMAINING_TASKS.md 更新
```
