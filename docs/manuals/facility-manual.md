# +TASTAS 施設管理者向け操作マニュアル

> **更新日**: 2025-01-07
> **対象者**: 施設の採用担当者・管理者

---

## はじめに

+TASTAS（タスタス）は、看護師・介護士向けの求人マッチングサービスです。
このマニュアルでは、施設管理者向けの機能と操作方法を説明します。

## 1. ログイン・アカウント管理

### 1.1 ログイン

1. `/admin/login` にアクセス
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

### 1.2 パスワード変更

1. 施設情報ページにアクセス
2. 「パスワード変更」をクリック
3. 現在のパスワードと新しいパスワードを入力

### 1.3 パスワードを忘れた場合

1. ログイン画面の「パスワードを忘れた方」をクリック
2. 登録メールアドレスを入力
3. 届いたメールのリンクからパスワードを再設定

## 2. 求人管理

### 2.1 求人一覧

サイドバーの「求人管理」をクリックすると、作成した求人一覧が表示されます。

**ステータス:**
| ステータス | 説明 |
|------------|------|
| 下書き | 作成中、未公開 |
| 公開中 | ワーカーに表示中 |
| 募集終了 | 締切済み |
| 完了 | 勤務完了 |

### 2.2 求人作成

1. 「新規作成」ボタンをクリック
2. 各項目を入力
3. 「公開」または「下書き保存」をクリック

**入力項目:**

| カテゴリ | 項目 |
|----------|------|
| 基本情報 | 求人タイトル、勤務時間（開始〜終了） |
| 勤務日 | カレンダーから日付を選択（複数可）、日別の募集人数 |
| 報酬 | 時給（日給は自動計算）、交通費 |
| 締切設定 | 勤務日の何日前まで応募可能か |
| 業務内容 | チェックボックスで選択、詳細を記入 |
| 資格要件 | 必要な資格を選択 |
| 服装・持ち物 | 必要な服装や持ち物を入力 |
| 移動手段 | 車OK、駐車場ありなどを選択 |
| こだわり条件 | 未経験歓迎、髪型自由などを選択 |
| マッチング方式 | 即時マッチング or 審査後マッチング |
| 責任者情報 | 担当者名、メッセージ、写真 |
| 画像 | 施設・職場の写真をアップロード |

### 2.3 マッチング方式の選択

**即時マッチング（デフォルト）:**
- ワーカーが応募すると自動的にマッチング成立
- 迅速に人員確保したい場合に最適

**審査後マッチング:**
- 「審査してからマッチング」にチェック
- 応募者を確認してから手動で採用/不採用を決定
- 応募者の選考を行いたい場合に最適

### 2.4 求人種別

| 種別 | 説明 | 対象者 |
|------|------|--------|
| 通常 | 一般公開の求人 | 全ワーカー |
| 勤務実績あり限定 | 過去に勤務したワーカー限定 | 施設で勤務経験者 |
| お気に入り限定 | 施設をお気に入り登録中のワーカー限定 | お気に入り登録者 |
| オファー | 特定ワーカーへの指名求人 | 指名したワーカー |

### 2.5 求人テンプレート

よく使う求人内容をテンプレートとして保存できます。

**テンプレート作成:**
1. サイドバーの「テンプレート」をクリック
2. 「新規作成」をクリック
3. テンプレート名と内容を入力
4. 保存

**テンプレート使用:**
1. 求人作成時に「テンプレートから作成」を選択
2. 使用するテンプレートを選択
3. 必要に応じて内容を編集

### 2.6 求人の編集・削除

**編集:**
1. 求人一覧から該当求人を選択
2. 「編集」ボタンをクリック
3. 内容を修正して保存

**削除:**
1. 求人一覧から該当求人を選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「削除」を選択

**注意:** 応募者がいる求人は削除できません。

## 3. 応募者管理

### 3.1 応募者一覧

サイドバーの「応募管理」をクリック。

**ビュー切り替え:**
- **求人から**: 求人ごとに応募者を確認
- **ワーカーから**: ワーカーごとに応募履歴を確認

### 3.2 応募の確認（審査ありの場合）

「審査後マッチング」を設定した求人の場合：

1. 応募一覧から該当の応募を選択
2. ワーカーのプロフィールを確認
3. 「マッチング決定」または「不採用」を選択

**確認できる情報:**
- 氏名、年齢、性別
- 保有資格
- 経験
- 自己PR
- 過去の勤務履歴
- 評価
- キャンセル率

### 3.3 ステータスの見方

| ステータス | 説明 |
|------------|------|
| 応募中 | 審査待ち |
| マッチング | 勤務確定 |
| 勤務中 | 現在勤務中 |
| 完了 | 勤務完了 |
| ワーカー辞退 | ワーカーがキャンセル（キャンセル率に影響） |
| 施設キャンセル | 施設がキャンセル |
| 応募取消 | 審査中に取り消し（キャンセル率に影響なし） |

### 3.4 キャンセル

マッチング成立後にキャンセルが必要な場合：

1. 該当の応募を選択
2. 「キャンセル」ボタンをクリック
3. 確認ダイアログで「キャンセル」を選択

**注意:** ワーカーにキャンセル通知が送信されます。

## 4. ワーカー管理

### 4.1 ワーカー一覧

サイドバーの「ワーカー管理」をクリック。

過去に応募・勤務したワーカーの一覧が表示されます。

**フィルター:**
- 職種（看護師、介護士など）
- 勤務ステータス
- 名前検索

### 4.2 ワーカー詳細

ワーカーを選択すると詳細情報が表示されます。

**表示される情報:**
- プロフィール
- 保有資格
- 経験
- 過去の勤務履歴
- 評価一覧
- キャンセル率

### 4.3 お気に入り・ブロック

**お気に入り登録:**
- 優秀なワーカーをお気に入りに登録
- お気に入り限定求人を出すことができる

**ブロック:**
- 問題のあるワーカーをブロック
- ブロックしたワーカーには求人が表示されない

## 5. メッセージ機能

### 5.1 メッセージ一覧

サイドバーの「メッセージ」をクリック。

ワーカーごとに会話がまとまって表示されます。

### 5.2 メッセージ送信

1. ワーカーを選択
2. メッセージを入力
3. 「送信」をクリック

**添付ファイル:**
- 画像: jpg, png, gif など
- 文書: pdf, doc, xlsx など
- 最大: 15MB/ファイル、3ファイルまで

### 5.3 自動メッセージ

以下のタイミングで運営から自動メッセージが送信されます：

- マッチング成立時
- 勤務前日リマインド
- 勤務完了・レビュー依頼

## 6. レビュー機能

### 6.1 ワーカーへのレビュー投稿

勤務完了後、ワーカーを評価できます。

1. ワーカー詳細画面の「レビューを書く」をクリック
2. 評価項目を入力
3. 送信

**評価項目:**
| 項目 | 説明 |
|------|------|
| 総合評価 | 5段階評価 |
| 出勤・時間厳守 | 時間を守れたか |
| スキル・技術 | 業務スキル |
| 業務遂行力 | 指示通りに業務をこなせたか |
| コミュニケーション | 報連相ができたか |
| 勤務態度 | 積極性・姿勢 |
| 良かった点 | 自由記述 |
| 改善点 | 自由記述 |

### 6.2 レビューテンプレート

よく使う評価文をテンプレートとして保存できます。

**作成方法:**
1. ワーカーレビュー画面でテンプレート管理を開く
2. 「新規作成」をクリック
3. テンプレート名と内容を入力

### 6.3 ワーカーからのレビュー確認

ワーカーからのレビューは「施設レビュー」で確認できます。

- 総合評価
- 良かった点
- 改善点

## 7. 施設情報の編集

### 7.1 基本情報

サイドバーの「法人・施設」をクリック。

**編集可能な項目:**
| 項目 | 説明 |
|------|------|
| 法人名 | 運営法人名 |
| 法人番号 | 13桁の法人番号 |
| 施設名 | 施設の名称 |
| 施設種別 | 特養、老健、デイサービスなど |
| 住所 | 郵便番号、都道府県、市区町村、番地 |
| 電話番号 | 連絡先電話番号 |
| 担当者 | 採用担当者名 |
| 挨拶文 | 施設紹介文（180文字まで） |

### 7.2 画像の設定

- 施設外観写真
- 施設内の写真
- 担当者写真

### 7.3 地図の更新

住所を変更した場合、地図は自動で更新されます。

手動で更新したい場合は「地図画像を更新」ボタンをクリック。

**注意:** マップピンの位置を微調整したい場合は、運営サポートまでお問い合わせください。

## 8. 通知について

### 8.1 通知の種類

| 通知 | タイミング |
|------|------------|
| 新規応募 | ワーカーが応募した時 |
| ワーカーキャンセル | ワーカーが予約をキャンセルした時 |
| 新着メッセージ | ワーカーからメッセージが届いた時 |
| 勤務リマインド | 勤務前日 |
| レビュー依頼 | 勤務完了後 |
| 募集枠充足 | 募集人数に達した時 |
| 締切間近 | 応募が少ない求人の締切が近い時 |

### 8.2 通知の受け取り方

**メール通知:**
- 登録メールアドレスに送信

**プッシュ通知:**
- ブラウザのプッシュ通知を許可してください

## 9. よくある質問

### Q. 求人を複製したい

**A.** 求人詳細画面の「複製」ボタンを使用するか、テンプレート機能を活用してください。

### Q. 応募者のキャンセル率はどこで確認できますか？

**A.** ワーカー詳細画面で確認できます。マッチング成立後にキャンセルした割合が表示されます。

### Q. 特定のワーカーにだけ求人を出したい

**A.** 「オファー求人」機能を使用してください。ワーカー詳細画面から「オファーを送る」で作成できます。

### Q. 過去に勤務したワーカーだけに求人を出したい

**A.** 求人作成時に求人種別を「勤務実績あり限定」に設定してください。

### Q. 応募が来ない

**A.** 以下を確認してください：
- 時給が相場より低くないか
- 募集期間が短すぎないか
- 求人タイトルが魅力的か
- 必要資格や条件が厳しすぎないか

### Q. 複数の担当者でアカウントを共有できますか？

**A.** セキュリティ上、アカウントの共有は推奨しません。複数担当者が必要な場合は運営までご相談ください。

### Q. 退会したい

**A.** 運営サポートまでお問い合わせください。未完了の予約がある場合は先にキャンセルしてください。
